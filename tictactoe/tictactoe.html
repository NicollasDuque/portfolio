TIC TAC TOE

My first project was a simple yet engaging Tic Tac Toe game, initially written in Python with a Tkinter interface.
The goal was to create a functional, user-friendly game where two players could compete on the same screen.
However, after completing the Python version, I decided to rewrite the game in JavaScript to make it more accessible, creating a pop-up window for easy trial and testing in a browser.
This transition posed several challenges, particularly in adapting the logic to JavaScript and ensuring smooth interaction between the user interface elements.
Through this project, I strengthened my problem-solving skills, refined my understanding of game logic, and gained hands-on experience with both Python and JavaScript.
Additionally, I became more proficient in working with HTML and CSS for layout and styling, and I deepened my knowledge of event-driven programming while developing the pop-up window interface.
Ultimately, this project helped me grow as a developer, honing both my technical and creative abilities.
